このリポジトリは、私のDXマネージャーとしての経歴、スキルセット、およびシステム開発実績を紹介するために構築したWebサイトのソースコードです。

✨ 特徴 (Features)

Bento Grid Design: 最新のトレンドを取り入れた、情報密度が高く視認性に優れたグリッドレイアウトを採用。

Fully Responsive: 物流現場や移動中でも確認できるよう、モバイル・タブレット・PC全てのデバイスに最適化。

High Performance: React等のフレームワークを使用せず、Vanilla JS (ES6+) と Tailwind CSS (CDN) のみで構築し、軽量かつ高速な読み込みを実現。

Enhanced UX:

History API連携: モーダル展開時にURLハッシュを変更し、ブラウザの「戻る」ボタンでの自然な閉じる動作を実装（SPAライクな挙動）。

Micro Interactions: スクロール時のフェードイン（Reveal）、ホバーエフェクトなど、心地よい操作感を提供。

🛠 技術スタック (Tech Stack)

このポートフォリオ自体も、私が持つフロントエンド実装スキルの一例です。

HTML5: セマンティックなマークアップとOGP設定によるSEO/SNS最適化。

CSS3 (Tailwind CSS): ユーティリティファーストCSSによる迅速なスタイリングと一貫性のあるデザインシステム。

JavaScript (Vanilla JS): 外部ライブラリへの依存を最小限（Lucide Iconsのみ）に抑えた、ネイティブなDOM操作とイベントハンドリング。

Fonts: Inter (英数字), Noto Sans JP (日本語) - 可読性と美しさを両立。
